Servlets

  • Servlet01: Hello World
    Write a HelloWorldServlet that prints out the text ‘HelloWorld’ when a get request is received. Put the code in a package be.yourcompany.servlets.
    The HelloWorldServlet shall react to following URL : http://localhost:8080/mywebapp/servlet/helloworldservlet
    (replace "mywebapp" with your context name)
  • Servlet02: Hello Name
    Write HelloNameServlet by making it accept a name which it prints out. Use the getParameter on the HttpRequestObject to obtain the name provided in the URL
    Your servlet shall react to following URL : http://localhost:8080/mywebapp/servlet/hellonameservlet?Name=myName
    (replace "mywebapp" with your context name)
